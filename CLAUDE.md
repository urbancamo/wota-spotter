# Project: WOTA Spotter

## Project Description
WOTA Spotter is an alerting and spotting application for the Wainwrights on the Air Amateur Radio Programme,
augmenting functionality on the main website https://wota.org.uk

## Tech Stack
- Frontend: Vue, TypeScript
- Backend: Node.js, Express
- Database: MySQL
- ORM: prisma
- Testing: Vitest

## Code Conventions
- We use Prettier for formatting
- ESLint for linting
- Functional components with hooks for React
- 2-space indentation
- camelCase for variables and functions
- PascalCase for components and classes

## Project Structure
- /src - Main source code
    - /components - Vue components
    - /assets - HTML assets
